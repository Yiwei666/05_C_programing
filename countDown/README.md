### date.c

这段代码的功能是根据用户输入的未来时间点（格式为：天-小时-分钟），实时计算并显示当前时间与未来时间点之间的剩余时间。程序首先获取用户输入的未来时间点，然后计算当前时间与未来时间点之间的时间差，并将时间差转换为小时、分钟和秒的格式。最后，程序每隔5秒更新并显示剩余时间。
